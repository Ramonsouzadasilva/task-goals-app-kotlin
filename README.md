# Task Goals

Aplicativo Android desenvolvido em **Kotlin** para gerenciamento de tarefas e metas.

## ✨ Funcionalidades

- Criar tarefas
- Editar tarefas
- Excluir tarefas
- Marcar tarefas como concluídas
- Criar metas
- Acompanhar o progresso das metas

## 🛠️ Tecnologias

- Kotlin
- Android SDK

## 🚀 Como executar

### Pré-requisitos

- Android Studio
- JDK 17 ou superior
